# Password Generator

A simple and secure password generator web app that helps you create strong, random passwords to keep your accounts safe.

## Live Demo

Try it out here: [Password Generator](https://capable-pothos-0ed0ef.netlify.app/)

## Features

- Generate random passwords with customizable length
- Includes uppercase, lowercase, numbers, and special characters
- User-friendly interface
- Instant password generation with a single click
- Responsive design for all devices

## How to Use

1. Select the desired password length.
2. Choose the types of characters to include (uppercase, lowercase, numbers, symbols).
3. Click the "Generate" button.
4. Copy your strong password and use it wherever you want!

- HTML5
- CSS3
- JavaScript
